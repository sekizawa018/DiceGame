## ルール<br>
- 二人で遊ぶサイコロゲーム
- サイコロを振って出た目がポイントになる
- 先に目標ポイントに到達したほうが勝利
- プレイヤーは自分のターンに何度でもサイコロを振ることができるが、
- サイコロの目が１だった場合、そのターンに得たポイントは没収され、ターンが相手に移る。
